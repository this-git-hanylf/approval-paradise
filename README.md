# approval-pakubuwono
Approval Mobile Pakubuwono



# Installation
```
$ yarn install .
```

# Run Android
> react-native start
```
$ yarn start
```
> react-native run-android
```
$ yarn android
```
# Run Ios
> react-native start
```
$ yarn start
```
> react-native run-ios
```
$ yarn ios
```

# Build Android Release
> gradlew app:assembleRelease
```
$ build-android-release
```
> aab release app:bundleRelease
```
$ build-aab-release
```
